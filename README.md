# MetaLens AI

MetaLens AI is an intelligent file analysis and digital forensics web application that extracts, analyzes, and visualizes metadata from uploaded images and PDF documents. It provides detailed file insights, privacy analysis, image forensics, and interactive visualizations through a modern cyber-themed interface.

## ✨ Features

- 📁 Image & PDF metadata extraction
- 📷 EXIF camera information (Make, Model, ISO, Exposure, Lens)
- 📍 GPS location detection with interactive map
- 📅 Creation & modification date analysis
- 🤖 AI-generated file summary
- 🛡️ Privacy risk & metadata quality analysis
- 🔍 Digital forensics checks
- 🎨 Dominant color palette extraction
- 📊 Image quality analysis
- 📄 PDF intelligence (pages, author, reading time, language)
- 📱 QR code detection
- 📈 Interactive file timeline
- 🆚 Compare two images
- 📤 Export reports (JSON, CSV)

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/MetaLens-AI.git
```

2. Open the project folder.

3. Open `metalens.html` in your browser.

No installation is required since the application runs entirely in the browser.
